# Customization

## Theme customization

Open **Settings → Theme** to customize:

- Primary and secondary accent colors
- Background and text colors
- Tile style (`solid` / `glass`)
- Background gradient type
- Layout spacing and border radius
- Typography base size and line height
- Preset switching (Linear Dark, Vercel Night, Figma Slate, Obsidian Glass)

Theme updates are applied live through CSS variables.

## Project customization

Open **Settings → Projects** to:

- create new projects
- delete custom projects
- switch project context from sidebar

## Tool customization

Open **Settings → Tools** to configure integration paths:

- MarkText executable
- MkDocs docs folder
- Joplin executable and optional notes export JSON
- Kanri executable and optional tasks export JSON
